AI CCTV 製造監控系統

把影像辨識、即時串流、告警與紀錄真正接到現場流程裡

這個專案是為生產線打造的自動化影像辨識系統，重點不只是辨識結果， 而是讓整體流程可以在現場穩定執行、持續監控並留下可追查的資料。

  • 基於 YOLO11/YOLOWorld/YOLOE26等以及結合HSV等傳統影像辨識 的自動化影像辨識系統，專為生產線設計，支援 RTSP 即時串流處理、ROI 區域監控、製程統計與異常偵測。
  • 多站點支援：不同製程節點可共用同一套監控架構。
  • 即時硬體聯動：透過 Modbus TCP 控制警示燈與現場告警訊號。
  • 自動化報表：提供班別統計（Day/Night）與 CSV 紀錄，方便稽核與追蹤。
  • API 整合：即時回報生產數據至 Web 系統，串接既有 WIP/管理平台。

System Context Flow

Training Pipeline
監控行為定義 → 製程場景盤點 → 影像資料擷取 → 抽幀與清洗 → 資料標註 → 模型訓練 → 監控邏輯開發 → 影片驗證 → True/False 校驗 → 現場上線
Inference Pipeline
擷取 CCTV 影像 → 影格轉換與推論 → 目標偵測結果輸出 → 邏輯規則判定 → 異常警報與 DB 記錄 → 異常片段回朔
Operational Outcome 使用者可透過 CCTV 警報系統與 CCTV 統計管理介面，結合 WIP System，達成異常行為即時告警、現場快速糾正與事後追蹤回溯。
