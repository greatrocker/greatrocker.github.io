AI CCTV 製造監控系統

把影像辨識、即時串流、告警與紀錄真正接到現場流程裡

這個專案是為 PCB 生產線打造的自動化影像辨識系統，重點不只是辨識結果， 而是讓整體流程可以在現場穩定執行、持續監控並留下可追查的資料。 [Read More]

  • 基於 YOLOv8/v10 的自動化影像辨識系統，專為 PCB 生產線設計。實現 RTSP 即時串流處理、ROI 區域監控、製程統計與異常偵測。
  • 多站點支援：DES, ENIG, OSP, VRS 等
  • 即時硬體聯動：Modbus TCP 控制警示燈
  • 自動化報表：班別統計 (Day/Night) 與 CSV 記錄
  • API 整合：即時回報生產數據至 Web 系統
